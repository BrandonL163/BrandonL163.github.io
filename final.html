Final Project

For my groups final project, me and my partner decided to create a spin off of the classic "rock paper scissors" game. We named our game "rock, star, heart. There is a main screen for our game that provides a start button and an instruction button, so if the user has never played the game before they can learn. When the user clicks the start button they will be sent to the game screen where they will be shown three buttons, rock, star and heart. The game plays the same as rock paper scissors as in the CPU will choose a random choice and the player clicks the button to select their choice. If the user wins round one, they will be sent to the second level. If the user wins again, they will be sent to the winning screen the congratulates them. If they lose, they will be sent to the losing screen where it will show what choice the computer picked.
